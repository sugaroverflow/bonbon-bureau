# Idea Pantry

The Idea Pantry is Bonbon Bureau's parking lot for future ideas.

Use it for things that are cute, promising, or talk-worthy, but not part of the immediate implementation path.

## Current pantry items

### Candyland

Future local factory / worker-hive environment for boring chores and utility jobs. Out of scope for the first Bureau scaffold.

### Jellybean Junction

Possible future connector between Bonbon Bureau and Candyland.

### Richer agent attribution

Maybe later: reusable GitHub PR templates, repo labels, or automation that fills the Bonbon Bureau PR block automatically.

## Rule

If an idea is exciting but would make the current Bureau harder to explain, put it here first.
